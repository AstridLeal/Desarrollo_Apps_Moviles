 FLUJO DE INFORMACIÓN
    ============================
    
    [PANTALLA] ← 1. Pide datos
         ↓
    [VIEWMODEL] ← 2. Procesa la petición
         ↓
    [USE CASE] ← 3. Aplica reglas
         ↓
    [REPOSITORY] ← 4. Obtiene datos
         ↓
    [DATOS] (API/Base de datos)
    
    ¡Y luego los datos regresan por el mismo camino! ⬆️
